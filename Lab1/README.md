# SJSU DATA 226 - Data Engineering

This repository contains coursework for DATA 226 at SJSU.

## Projects
### 1. Homework 5: Weather Forecast ETL
- **Description**: An ETL pipeline that fetches weather data from Open-Meteo API and loads it into Snowflake.
- **Key Features**:
  - SQL Transactions (BEGIN, COMMIT, ROLLBACK).
  - Full Refresh pattern for data loading.
  - Containerized using Docker & Apache Airflow.
